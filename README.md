

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start


# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
Runs the app in the development mode.\
Open [http://localhost:4200/graphql](http://localhost:4200/graphql) to view it in the browser.
## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
